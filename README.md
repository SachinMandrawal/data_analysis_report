# Data Analysis Report

This is a project report on **NumPy** using Jupyter Notebook, focusing on:
- *Array Creation Routines*
- *Array Manipulation Routines*

## Overview
This project explores fundamental NumPy operations for array handling and manipulation.

## Topics Covered
### Array Creation
- Basic array creation methods
- Arrays from existing data
- Numerical ranges
- Matrix creation utilities

### Array Manipulation
- Changing array shape
- Transpose operations
- Changing array dimensions
- Joining and splitting arrays

## Tools Used
- Python
- NumPy
- Jupyter Notebook
